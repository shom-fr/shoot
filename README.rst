SHom Ocean Objects Tracker
==========================

Detect and track ocean objects like eddies, fronts, river plumes and solitary waves.

Installation
------------

From sources::

   pip install .

Documentation
-------------

The shoot documentation:
https://apps.shom.fr/shoot
